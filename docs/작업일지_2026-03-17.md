## 5. Google OAuth 설정 안내

**Client ID**: `230374755335-4hd0g37rnamvt4vtagtlf65rfe3o537b.apps.googleusercontent.com`

### 적용 위치
코드가 아닌 **Supabase 대시보드**에 등록 필요 (Supabase가 OAuth 중계)

### 설정 순서
1. Supabase Dashboard → `Authentication` → `Providers` → **Google** 활성화
2. Client ID 입력 (위 값)
3. Client Secret 입력 (Google Cloud Console에서 확인)
4. Google Cloud Console → OAuth 클라이언트 → 승인된 리디렉션 URI 추가:
   ```
   https://heumjgskjglarxcwqcmp.supabase.co/auth/v1/callback
   ```
5. (배포 후) Supabase → URL Configuration:
   - Site URL: Railway 배포 URL
   - Redirect URLs: `https://{railway-url}/auth/callback`
